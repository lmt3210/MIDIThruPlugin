Welcome

MIDIThru is a simple MIDI processor audio unit that will pass through all MIDI traffic except system messages (system exclusive, sequencer start, MIDI clock, active sensing, etc.) unchanged.

Usage

Insert MIDIThru into a DAW MIDI track or instrument track. A red indicator will flash to indicate activity when MIDI data is received.

Not all DAWs support MIDI effect plugins (including Ableton Live). Consequently, MIDIThru will not appear in the DAW's list of available plugins.
